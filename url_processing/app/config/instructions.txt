Route : 	/extract_page
requests.post('http://127.0.0.1:5000/extract_page', json=data)
Input : 	data object in json format is returned
			data['url'] => input url
Output : 	final object in json format is returned
			if url sent is a webpage:
				final['type'] = 'html' (denotes type of url) 
				final['text'] = clean text from the webpage
				final['headings'] = headings extracted
				final['links'] = links extracted
				final['tables'] = tables extracted
			if url sent is any pdf (with name of pdf in the url):
				final['type'] = 'pdf'
				final['path'] = path to the pdf file
			if any errors:
				return object is {"error":"error type"}
